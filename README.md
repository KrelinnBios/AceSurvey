## 项目简介

AceSurvey 是一个面向无性恋社群的静态问卷与卡片生成工具。当前版本将问卷填写、浏览器草稿保存、卡片预览和 PNG 导出整合在同一个页面中，帮助访问者梳理无性恋谱系身份、发现经历、相关认知、性行为态度与社群期待等基本信息。

项目只描述当前实际提供的功能，不提供远程提交接口。

## 功能概览

- 问卷填写：提供完整的问卷填写界面和分组题项。
- 题项范围：覆盖无性恋谱系类型、年龄、发现经历、类型认同理由、Ace 谱系认知、性行为态度、相关社群议题看法和社群期待。
- 草稿保存：填写内容会保存到浏览器本地存储，刷新或退出页面后可继续编辑。
- 卡片导出：基于 html2canvas 生成完整问卷卡片，默认文件名为 `ace_community_survey.png`。
- 移动适配：针对移动浏览器和 QQ 内置浏览器做了输入、滚动、预览与长按保存优化。

## 界面预览

当前界面为单页问卷工具，由问卷填写区、卡片预览区和导出入口组成，可通过在线地址或本地 `index.html` 预览。

## 使用方式

### 在线访问

访问 <https://acesurvey.pages.dev/>。

### 本地使用

直接用浏览器打开 [index.html](./index.html)。

页面会加载 Google Fonts 和 html2canvas CDN；如需完整字体与图片导出体验，请保持网络可用。

### 线上部署

本项目是静态网页，可部署到 GitHub Pages、Cloudflare Pages、Vercel、Netlify 等静态网站托管平台。

## 技术信息

- 技术栈：HTML、CSS、JavaScript。
- 外部资源：Google Fonts 与 html2canvas CDN。
- 数据存储：通过浏览器 `localStorage` 保存草稿，不提供远程提交接口。
- 图片导出：通过 html2canvas 在浏览器端生成 PNG。

## 内容边界

- 本项目内容仅供社群问卷填写、自我梳理和身份表达辅助，不构成医疗、心理、法律或其他专业建议。
- 页面中的问题设置不代表对任何身份、立场或经历的诊断、评估或价值判断。
- 本项目不提供远程提交接口；访问者在本地填写、暂存、截图或导出的个人回答不属于本仓库预设内容，其保存、分享和使用方式由访问者自行决定。
- 如果将导出的问卷卡片用于社群审核或公开分享，请先确认其中不包含不愿公开的个人信息。
